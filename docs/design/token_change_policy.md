# Token Change Policy

1. Сначала менять semantic alias `--ds-*`, а не raw Telegram variable.
2. Любое изменение typography или spacing проверять на `320px`, `768px`, `1280px`.
3. Любое изменение accent behavior перепроверять в светлой и темной Telegram themes.
4. Если меняется радиус или elevation карточек врача, обязателен visual QA home/list/detail.
