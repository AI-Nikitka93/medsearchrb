# Brand Rules

## Допустимо
- Спокойная, уверенная типографика.
- Telegram-native цвета через `--tg-theme-*`.
- Один явный акцент на экран.
- Подчеркивать trust signals через форму, а не крикливый цвет.

## Недопустимо
- Собственные фоновые HEX-цвета.
- Иллюзия, что продукт сам записывает к врачу.
- Рекламный тон скидочного агрегатора.
- Холодный "медицинский кабинет" без характера.
