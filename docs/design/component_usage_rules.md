# Component Usage Rules

## Search Field
- Один на экран как основной вход.
- Не дублировать вторым поиском ниже fold.

## Filter Chips
- До 5 видимых чипов без переполнения.
- Selected state должен отличаться не только цветом.

## Doctor Card
- Не больше 6 смысловых элементов в compact mode.
- Имя врача всегда выше клиники по иерархии.

## Sticky CTA
- Использовать только на detail screen.
- Нельзя показывать две конкурирующие primary CTA одновременно.
